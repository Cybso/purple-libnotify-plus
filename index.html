purple-libnotify+

A libnotify dispatching plugin

purple-libnotify+ provides events dispatching for purple-events through libnotify.
It was inspired by Pidgin-libnotify and Guifications.

Dependencies

  • Pidgin (2.6.0 or later)
  • purple-events
  • libnotify

Releases download

Visit the downloads page to find releases (tar.xz) and automatic GitHub tag tarball/zip.

Git repository

The repository is hosted on GitHub: purple-libnotify+ Git repository

You can clone the project with Git by running:
$ git clone git://github.com/sardemff7/purple-libnotify-plus
